# TV Volume Widget - 设计文档 (已确认)

## 一、项目概述

### 1.1 背景与痛点

用户通过 HDMI 将 Windows 电脑连接电视，用于大屏游戏/观影。每次切换音频输出设备需要回到电脑显示器前操作系统托盘小图标，在电视上完全无法操作，体验极差。

### 1.2 核心目标

开发一个**轻量级、美观、大屏友好**的 Windows 桌面应用，让用户可以：
- 在电视大屏上清晰看到所有音频输出设备
- 一键切换音频输出设备
- 直观调节音量
- 无需回到电脑显示器前操作

### 1.3 一句话描述

> 一个为电视大屏设计的音频设备切换器，大字体、大图标、触控友好、颜值在线。

## 二、已确认的技术方案

### 2.1 框架：Flutter (Windows Desktop)

### 2.2 依赖清单

| 包 | 用途 |
|---|---|
| `win32` | Windows API 绑定（FFI 基础） |
| `flutter_volume_controller` | 音量控制 |
| `hotkey_manager` | 全局热键 |
| `bitsdojo_window` | 无边框窗口、置顶、透明度 |
| `tray_manager` | 系统托盘 |
| `shared_preferences` | 设置持久化 |

### 2.3 音频后端

- `IMMDeviceEnumerator` 枚举音频端点
- `IAudioEndpointVolume` 控制音量
- `IPolicyConfig` 切换默认设备
- `PKEY_AudioEndpoint_FormFactor` 检测设备类型

### 2.4 设备类型检测

通过 `PKEY_AudioEndpoint_FormFactor` 属性 + 设备名称关键词判断：

| FormFactor 值 | 类型 | 图标 |
|---|---|---|
| 1 | Speakers（音箱） | speaker_icon |
| 3 | Headphones（耳机） | headphone_icon |
| 5 | Headset（头戴耳麦） | headset_icon |
| 9 | DigitalAudioDisplayDevice（HDMI 电视） | tv_icon |
| 8 | SPDIF（数字音频） | digital_icon |

### 2.5 窗口行为

- 常规窗口，支持拖拽移动
- 支持置顶显示
- 支持透明度调节（20%-100%）
- 最小化到系统托盘

### 2.6 全局热键

- `Ctrl+Alt+Up` / `Ctrl+Alt+Down`：音量调节（步进 1%）
- `Ctrl+Alt+M`：静音切换
- `Ctrl+Alt+D`：显示/隐藏窗口
- 全屏应用时不呼出窗口（通过检测前台窗口尺寸实现）

### 2.7 分发方式

- 便携版：Flutter 构建后的文件夹（~25-30MB），双击 exe 即用
- 可选：Enigma Virtual Box 打包为单个 exe

### 2.8 多显示器

- 记忆上次窗口位置
- 首次启动默认主显示器

### 2.9 不实现的功能

- 麦克风等输入设备管理

## 三、UI/UX 设计

### 3.1 设计原则

1. **大屏优先**：所有文字 ≥ 24px，图标 ≥ 48px，确保 3-5 米距离可辨识
2. **简洁至上**：一个页面完成所有操作，无需导航
3. **触控友好**：按钮最小点击区域 60×60px，间距充足
4. **视觉舒适**：莫兰迪色系，低饱和度，不刺眼

### 3.3 莫兰迪色系主题

提供 5 种预设莫兰迪主题色（低饱和度、高级灰调）：

| 主题名 | 主色调 | 背景色 | 效果 |
|--------|--------|--------|------|
| 烟粉 (Dusty Pink) | #D4A5A5 | #2C2C2C | 温柔、放松 |
| 雾蓝 (Misty Blue) | #7FAABE | #2C2C2C | 冷静、专注 |
| 灰绿 (Sage Green) | #9CAF88 | #2C2C2C | 自然、舒适 |
| 暖杏 (Warm Apricot) | #D4A574 | #2C2C2C | 温暖、活力 |
| 藕紫 (Lavender Grey) | #9B8EA8 | #2C2C2C | 优雅、安静 |

### 3.4 交互设计

- **音量调节**：大尺寸滑块，支持鼠标拖拽和滚轮调节，键盘步进 1%
- **设备切换**：点击设备卡片即切换，切换后自动高亮+动画反馈
- **窗口行为**：
  - 支持置顶显示
  - 支持最小化到系统托盘
  - 支持透明度调节（20%-100%）
  - 窗口可拖拽移动
